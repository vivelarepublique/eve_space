# EVE Space

## What is it?

****EVE space**** is an information gathering tool and game aid for CEVE (EVE Serenity).

## Release version

No yet.

## Start

```bash
git clone https://github.com/vivelarepublique/eve_space
cd eve_space
npm install
npm run start
```
